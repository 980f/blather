ZK-4KX Adjustable power supply controller

Designed for solar power system battery chargers

  1. LCD can display input/output voltage, output current/output power/output capacity/output time;
  2. Digital control of parameters, retained while power is off.
  3. Protection against input polarity reversed
  4. Protection against reversed connection to battery
  5. Configurable to power up on or off
  6. Multiple configurable protection mechanisms.
    When the working parameters of the module exceed the protection threshold, the output will be turned off.
  7. Output ripple is small and PI filter is available (? available or just compatible)
  8. Good heat sink
  9. Soft start (with high power and load module may fail when starting)

 Specifications:

Input voltage: 5.0-30v below 8V power is limited
Output voltage: 0.5-30v
Output current: 3A 4A with enhanced heat dissipation
A short from output to input is not protected by this!
Output power: 35W 50W with enhanced heat dissipation
Voltage display resolution:  0.01V
Current display resolution: 0.001A (1 mA)
Conversion efficiency: about 88% Module will underperform if input power is not adequate
Operating frequency: 180 kHz
Dimensions: 79mm x 43mm x 26mm cutout 76 x 41

Operation

Switch displayed parameters 

From the normal interface, press SW to switch the lower line of the display between: current A, power W, capacity Ah, and time h.
Long press SW button to switch the upper display content between input voltage IN output voltage OUT.

Set output voltage or current

From the normal state press U/I button once for voltage, again for current.
Short clicks of the knob selects the resolution of the knob, shown by flashing the corresponding digit.
After setting, press U/I button to return to the normal interface.
Module will automatically return to the normal interface after 10 seconds of no user activity.

Set other limits

From the normal state long press U/I. Pressing SW will cycle through the options, see Control Settings table for the mnemonic for each feature.
For numerical values short presses of the knob cycles through the digits, rotate knob to change value.
For items that have an OFF option you must long press the rotary knob to toggle ON vs OFF.
Long press U/I to return to normal state.
