 Operation

Switch display parameters

in the normal state press SW to switch the 2nd line content between current A | power W | capacity Ah| time h.
Long press SW button to switch the 1st line content between input voltage and output voltage.

Setting limits

From the normal state press the U/I button to enter settings mode.
One digit of the value is flashing, rotate the encoder left and right to adjust it.
Short press the rotary encoder to choose which digit  to set.
First value is voltage, press U/I button again to edit constant (max) current.
Press U/I again to return to normal state, which will happen automatically after 10 seconds of inaction.

Other settings

In the normal state long press U/I to enter the parameter setting state.
Press SW to cycle through the parameters.
Press rotary encoder for toggles (enables) rotate for threshold value.
After settings long press U/I to return to the normal state.

Et Cetera

  1. Shorting between input IN and output OUT of the module is bad, at a minimum the current limit function will fail.
  2. Ensure that the power capacity of the input supply exceeds the power draw of the output load!
  3. To output at maximum capacity the input voltage should be above 8V. When the input voltage is 5V, the output power is limited to about 15W. The maximum module current value is 4A, subject to the maximum output power, e.g.  at 17V the current should not be greater than 2A.
  4. When this module is used over 3A and 35W, you must increase its own heat dissipation!!!
  5. The module input undervoltage protection function, which is defaulted to about 4.8v, will automatically disconnect the output, but beware of the voltage drop on the cable, the threshold is at the module end not your source and at high currents will trip apparently early.
